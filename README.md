# JubiBot

Discord Command Bot.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'jubibot', github: 'jubishop/jubibot'
```

You will need these dependencies, too:

```ruby
gem 'csv'
gem 'discordrb'

gem 'core', github: 'jubishop/core'
gem 'rstruct', github: 'jubishop/rstruct'
```

And then execute:

```sh
$ bundle install
```
