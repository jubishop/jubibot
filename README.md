# JubiBot

Discord Command Bot.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'jubibot', github: 'jubishop/jubibot'
```

And then execute:

```sh
$ bundle install
```

## Usage

See the code in [RLBot](https://github.com/jubishop/RLBot) for example use.
